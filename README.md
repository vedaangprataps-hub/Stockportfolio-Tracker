# 📈 Stock Portfolio Tracker

A simple command-line Stock Portfolio Tracker built using Python. This application allows users to enter stock names and quantities, calculates the total investment value using predefined stock prices, and optionally saves the portfolio report to a text file.

---

## 📌 Features

- User inputs stock names and quantities
- Uses a predefined dictionary of stock prices
- Calculates the total investment value
- Displays a detailed portfolio summary
- Option to save the report as a `.txt` file
- Beginner-friendly and easy to understand

---

## 🛠️ Technologies Used

- Python 3

## 🚀 How to Run

1. Clone this repository:

```bash
git clone https://github.com/YOUR-USERNAME/STOCK-PORTFOLIO-TRACKER.git
```

2. Navigate to the project folder:

```bash
cd STOCK-PORTFOLIO-TRACKER
```

3. Run the program:

```bash
python stock_tracker.py
```

---

## 🎯 How It Works

1. The program displays a list of available stocks and their prices.
2. The user enters the stock name and quantity.
3. The program calculates the investment value for each stock.
4. A portfolio summary is displayed.
5. The user can save the report to a text file.
